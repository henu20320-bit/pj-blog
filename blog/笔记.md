# 网络协议分析与攻防技术 · 第四讲

## 一、网络空间安全基础
### 1. 网络空间定义
- **构成要素**：
  - **设施**：互联网、通信系统、计算机系统、工控系统等
  - **数据**：可存储/传输的声光电磁信号
  - **用户**：网络角色（账户、软件、设备）
  - **操作**：产生信号、修改状态等行为
- **安全属性（CIA 三元组）**：
           
| 属性    | 核心目标        | 保护措施                                           |
| ----- | ----------- | ---------------------------------------------- |
| 机密性   | 物理安全、电磁对抗   | 信息加密、信息划分密级、防止硬件辐射泄露                           |
| 完整性   | 协议安全、网络对抗   | 严格控制对系统中数据的写访问，只允许许可的当事人进行更改。                  |
| 可用性   | 密码破解、通信安全   | 在坚持严格的访问控制机制的条件下，为用户提供方便和快速的访问接口，提供安全的访问工具<br> |
| 不可否认性 | 终端安全、信息传播对抗 | 数字签名、可信第三方认证技术                                 |

### 2. 安全作用空间
| 层级  | 安全焦点        |
| --- | ----------- |
| 硬件层 | 物理安全、电磁对抗   |
| 代码层 | 协议安全、网络对抗   |
| 数据层 | 密码破解、通信安全   |
| 应用层 | 终端安全、信息传播对抗 |

## 二、网络战（Cyber Warfare）
### 1. 核心特征
- **隐蔽性与匿名性**：通过跳板服务器/僵尸网络隐藏身份
- **低成本高收益**：瘫痪关键基础设施代价远低于传统战争
- **非对称性**：中小组织可对强国发起有效攻击
- **跨界融合**：结合舆论战、心理战放大社会动荡

### 2. 典型攻击形式
| 类型               | 目标案例                              | 真实事件                                |
|--------------------|---------------------------------------|-----------------------------------------|
| 关键基础设施攻击   | 能源/交通/医疗系统                    | 2015 乌克兰电网瘫痪；2021 美输油管道勒索  |
| 军事与情报窃取     | 国防网络/武器系统                     | 2010 震网病毒破坏伊朗核设施             |
| 经济战与工业间谍   | 企业核心技术/供应链                   | APT 41 窃取商业机密；2023 中国车企数据泄露 |
| 舆论操控           | 社交媒体/选举系统                     | 2016 俄干预美大选；2022 印尼网络水军     |

### 3. 技术趋势
- **AI 自动化攻击**：生成钓鱼内容、自主决策攻击路径
- **量子计算威胁**：破解 RSA/ECC 等加密算法
- **5 G/物联网漏洞**：智能城市设备成新攻击面
- **太空网络战**：卫星通信系统遭干扰（如星链）
- **生物技术融合**：攻击基因数据库/医疗设备

## 三、网络攻击
### 1. 攻击分类
| 分类维度       | 类型                | 特点                                      |
|----------------|---------------------|-------------------------------------------|
| **攻击来源**   | 外部攻击            | 来自系统外部（如黑客入侵）                |
|                | 内部攻击            | 合法用户越权访问                          |
|                | 行为滥用            | 特权用户滥用权限                          |
| **攻击影响**   | **被动攻击**        | 监听流量/内容（如网络嗅探）               |
|                | **主动攻击**        | 主动破坏系统（如数据篡改/身份假冒）       |

### 2. APT 攻击（高级持续性威胁）
- **特点**：
  - **长期性**：持续监控目标，保持长期访问权
  - **定向性**：针对高价值组织（政府/军工/科研）
  - **复杂性**：定制化攻击工具，挖掘系统漏洞
- **2024 年中国受 APT 影响行业 TOP 5**：
  政府机构 > 教育 > 科研 > 国防军工 > 交通运输

## 四、网络防护
### 1. 安全模型
| 模型          | 核心要素                              | 应用场景               |
|---------------|---------------------------------------|------------------------|
| **PDRR**      | 防护→检测→响应→恢复                   | 动态安全防护循环       |
| **IATF 纵深防御** | 保护网络/边界/计算环境/支撑基础设施   | 多层次安全措施         |

### 2. ISO 安全体系

| 组件             | 内容                                                                 |
|------------------|----------------------------------------------------------------------|
| **5 类安全服务**  | 鉴别、访问控制、数据机密性、数据完整性、抗抵赖                       |
| **8 种安全机制**  | 加密、数字签名、访问控制、数据完整性保护、认证交换、流量填充、路由控制、公证 |
| **普遍性机制**   | 安全审计、事件检测、安全恢复等                                       |

### 3. 安全技术演进
| 代际     | 核心理念                          | 关键技术                          |
|----------|-----------------------------------|-----------------------------------|
| 第一代   | 边界防护                          | 防火墙、访问控制、加密            |
| 第二代   | 检测+恢复                         | IDS/IPS、备份容灾                 |
| 第三代   | **入侵容忍**                      | 系统在受攻击时维持关键功能        |
